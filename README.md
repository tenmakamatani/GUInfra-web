# GUInfra-web

Web platform of GUInfra

## How to use

### Clone and Setup
```
$ git clone git@github.com:tenmakamatani/GUInfra-web.git
$ cd GUInfra-web
$ yarn install
```

### Development

```
$ yarn dev
```

### Production

```
$ yarn start
```
