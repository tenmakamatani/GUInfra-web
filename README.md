# GUInfra-web

Web platform of GUInfra

## Environment

- node v10.21.1
- npm v6.14.6

## How to use

### Clone and Setup
```
$ git clone [THIS REPOSITORY]
$ cd GUInfra-web
$ yarn/npm install
```

### Development

```
$ yarn/(npm run) dev
```

### Production

```
$ yarn/(npm run) start
```
